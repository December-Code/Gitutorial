# Git Tips

## Git config
### setting
```
global
git config --global user.name "your name"
git config --global user.email "your@email"

local
git config --local user.name "your name"
git config --local user.email "your@email"
```

### list
```
global
git config --gobal --list

local
git ocnfig --local --list
```
